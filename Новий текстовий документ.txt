Мій перший репозиторій
Git - система контролю версій файлів .
Статуси здійснені змінені поетапно не відстежуються. 
Команда add використовується для додавання файлів.
Команда статус використовує для перевірки статусу файлів.
Команда commit використовується для додавання файлів у локальний репозиторій.
Команда init використовується для ініціалізації н проекту
У даній команді ми повинні написати інструкцію, що ми змінюємо у commit.
Для того, щоб побачити список commit , мені потрібно використати "git log --reflog"
Команда перевірки дозволить відпустити зміни у файлі в потрібну вітку
